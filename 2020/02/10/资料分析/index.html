资料分析还是要多做题，将方法运用到题目中，加深印象，找到感觉。 😤

解题方法

速算技巧

截位直除

  • 截位：四舍五入保留

    • 如：678999，截两位 => 68
    • 如：0.67899，截两位 => 0.68

  • 截多少位？看选项

    • 首位不同：如 A.11 B.21 C.31 D.41
      • 选项差距大，截两位
    • 首位相同：如 A.11 B.21 C.25 D.41
      • 计算大小最接近两项的次位差（次位差：第二位的差；如：25 与 21 的次位差为 5 - 1 = 4）
        • 次位差 > 首位 ：选项差距大，截两位
        • 次位差 <= 首位 ：选项差距小，截三位

  • 怎么截？看算式

    • 一步除法（只有一个除号）：只截分母
      • $ A/B $ $ A+B/C $ $ A/B+C $
      • 如：$ 32345/26189 $ $ 4286/(1+0.21) $ $ (4286-172)/(1+0.21) $
    • 多步除法：分子、分母都截
      • 如：$ (71774/3658) / (12482/4760) $
  • 若遇到分母相加且计算量过大，那么也可以先截后加，相加后再截
    • 如：$ 71182/(71182+64222) $ 根据选项，我们需要截取三位
      • 71182 => 712 64222 => 642
      • 712 + 642 = 1354 => 135
      • $ 71182/135 $
  • 若选项存在约十倍关系时，考虑数量级（小数点、位数、单位）
    • 如：A.10 B.0.1 C.100 D.0.01

比较类

  • 一大一小，直接看：分子大的分数大
    • 如：$ 6/7 > 5/8 $ 6 > 5 (大) 7 < 8（小）为一大一小
  • 同大同小：
    • 如：$ 36/21 与 77/41 $ 36 < 77（小） 21 < 41（小）为同小
    • 竖着直接除
    • 横着看速度（变化倍速）：谁快谁牛气，慢的看成一
      • 如：$ 36/21 与 77/41 $
        • 分子：77 是 36 的两倍多一点；分母：41 是 21 的不到两倍
        • 慢的看成一：所以将它们的分母看成一
        • 所以 36 < 77，即$ 36/21 < 77/41 $

快速找数

  • 内容：纯文字型材料、图标型材料
  • 要求：
    • 掌握结构阅读法，会找关键词
    • 对于图标型材料会识别关键信息

文字材料

  • 特点：数据多，相近词多
  • 方法：认清结构、找到关键词、一一对应
  • 注意：
    • 标记段落主题词，与题干进行匹配
    • 注意相近词、时间、单位等

图表材料

  • 类型：柱形图、饼形图、表格
  • 方法：
    • 看表头三要素：时间、主体、单位
    • 有注释，一定要看注释

  • 饼状图：

    • 构图原则：12点钟方向开始，顺时针依次排布

  • 注意：

    • 表格材料
      • 横纵题目、标题、单位、备注
    • 图形材料
      • 标题、单位、图列

综合材料

  • 不同类型材料之间的关系、材料结构

坑点

  • 表格材料，“总计”坑
  • 单位坑（民航、人口）

题型技巧

基期与现期

  • 作为对比参照的是基期，而相对于基期比较的是现期
    • 时间来讲：过去的是基期；现在的是现期
    • 对象来讲：被比较的是基期
      • 如：马云比我有钱
        • 马云是现期
        • 我是基期
  • 增长量：现期 - 基期
  • 增长率（r）：$ (现期-基期)/基期 $

题型

基期量

  • 给现在，求以前某时期的值
  • 计算公式：
    • 基期量：现期 - 增长率
    • 基期率：$ 现期/(1+增长率) $
  • 速算公式
    • $ |r|>0.5 $，截位直除
    • $ |r|<= 0.5 $，化除为乘
      • 化除为乘：
        • 应用：求基期，选项差距小，$ |r|<=0.5 $
        • 方式:
          • $ A/(1-r)≈ A+A*r $
          • $ A / (1 + r) ≈ A-A*r $
  • 口诀
    • 读问题，看时间;
    • 判题型，找数据；
    • 列式子，先别算；
    • 要想快，看选项；

基期和差

  • 先观察现期坑，排除；再看大小关系，选择
  • 截位直除

现期量

  • 题型识别：给现在，求后面某时期的值
  • 计算公式：
    • $现期量 = 基期量 + 增长量$
    • $现期量 = 基期量 * (1 + r)$

同比和环比

  • 同比：与上年同期相比
  • 环比：与紧紧相邻的上一统计周期相比（月环比、季度环比）

一般增长率

  • 简单计算类：高减低加
  • 给具体量：
    • 识别：增长 / 下降 + %、几成、增长速度、增长幅度
    • 公式：$ r = 增长量/基期量 = 增长量/(现期-增长量) = (现期 - 基期)/基期 $
    • 速算：截位直除或估算
  • 增长率比较
    • 识别：增速（增幅）最快 / 最慢；增长率最高 / 最低
    • 比较方法：$ r = 增长量/基期量 = (现期 - 基期)/基期 = 现期/基期 - 1 $
      • 当$ 现期/基期 = 2^+ $，用$ 现期量/基期量 $比较
      • 当$ 现期/基期 = 1^+ $，用$ 增长量/基期量 $比较
    • 速算技巧：分数比较

增长量

  • 题型识别：增长 + 具体单位
    • 例：2017年比2016年增加了多少亿元
  • 计算公式：$ 增长量 = 现期 - 基期 = 基期 r = 现期/(1+r)r $

年均增长量

  • 识别：年均 + 增长 + 单位
  • 公式：$ 年均增长量 = (现期量 - 基期量) / 年份差 $
  • 注：现期、基期及年份差的确定和年均增长率相同

已知现期、增长率、求增长量

  • 公式：$ 增长量 = 现期/(1+r)*r $ 太复杂
    • 增长率百化分，$ |r| = 1/n $
    • $ 增长量 = 现期/(n+1)；减少量 = 现期/(n-1) $
  • 例：2016年总收入是100万元，同比增长了33.3%。求：2016年与2015年相比总收入增长了多少万元？
    • 增长率：33.3% => 百化分：$ r=1/3 $
    • 增长量：$ 100/(3 + 1) = 25$
  • 【百化分】
    • $ 1/2 = 50\%$ $1/4 = 25\%$ $1/8 = 12.5\%$ $1/16 = 6.25\%$
    • $ 1/3 \approx 33.3\% $ $ 1/6 \approx 16.7\% $ $ 1/12 \approx 8.3\% $
    • $ 1/5 = 20\% $ $ 1/10 = 10\% $ $ 1/20 = 5\% $
    • $ 1/7 \approx 14.3\% $ $ 1/14 \approx 7.1\% $
    • $ 1/9 \approx 11.1\% $ $ 1/11 \approx 9.1\% $
    • $ 1/13 \approx 7.7\% $ $ 1/15 \approx 6.7\% $
    • $ 1/17 \approx 5.9\% $ $ 1/18 \approx 5.6\% $ $ 1/19 \approx 5.3\% $
  • 增长率百化分之放缩法
    • 利用与背过的百分数的倍数关系，实现百化分
    • 2.5% = 1/40；1.43% = 1/70；67% = 1/1.5；22% = 1/4.5
  • 增长率百化分之取中法
    • 如果遇到百分数左右难取舍，且选项接近，取中即可
    • 18.5% = 1/5.5；15.4% = 1/6.5；28.7 = 1/3.5

增长量比较

  • 题型识别：增长最多 / 最少
  • 考察形式
    • 给现期量、基期量：增长量 = 现期量 - 基期量
    • 给现期量、r：
      • 大大则大
      • 百化分计算

比重

比重中的饼图问题

  • 直接给总量：找特殊值
    • 如：总量：100、四川：25 => 那么就有特殊值 1/4
  • 没有直接给总量：
    • 大小
    • 倍数

现期比重

  • 题型识别：…占…的比重
    • 如：某班总共500人，其中男生100人，求男生占全班总人数的比重
  • 公式：$ 比重 = 部分/总体 = 占前/占后$
  • 考察形式
    • 已知部分和总体，求比重
    • 已知部分和比重，求总体
    • 已知总体和比重，求部分
  • 速算技巧：截位直除
  • 概念引申：
    • $ 增长贡献率=部分增量/总体增量 $
    • $ 利润率=利润/收入 $

基期比重

  • 提醒识别：问题时间在材料之前，占、比重

    • 例：2017年我的家庭总收入B万元，同比增长b，

      ​ 2017年我个人收入A万，同比增长a，

      求：2016年我的收入占家庭总收入的比重

  • 计算公式：$ A/b * (1 + b)/(1 + a) $

    • A：分子（部分的现期量） B：分母（整体的现期量）
    • a：分子的增长率 A：分母的增长率

  • 速算：

    • 截位直除
    • 先算现期比重，再看$ (1+b)/(1+a) $ 大于1或者小于1，结合选项排除答案

现期与基期比重比较与计算（两期比重）

  • 比较 ：升降

    • 提醒识别：两个时间 + 比重

      • 【识别】2013年1~9月，苏中工业用电量占江苏省工业用电量总量的比重与去年相比：

        A.提高 B.降低 C.不变 D.无法判断

    • 计算公式：$ 现期比 - 基期比 = A/B - A/B(1+b)/(1+a)=A/B(a-b)/(1+a) $

    • 升降判断：

      • a > b，比重上升
      • a < b，比重下降
      • a = b，比重不变
      • a：分子（部分量）的增长率；b：分母（总体量）的增长率

  • 上升/下降几个百分点

    • 提醒识别：两个时间 + 比重

      • 【识别】2015年一季度，该省园区企业上缴税金占主营业务收入的比重比上年同期：

        A.上升了0.1个百分点 B.上升了3.1个百分点

        C.下降了0.1个百分点 D.下降了3.1个百分点

    • 计算公式：现期比 - 基期比

    • $A/B-A/B(1+b)/(1+a)=A/B(a-b)/(1+a)$

    ​ $=A/B1/(1+a)(a-b)<|a-b|$

    • 解题步骤：
      • 判定方向（a > b，上升；a < b，下降）
      • 定大小：小于 | a - b |
      • a：分子（部分量）的增长率；b：分母（总体量）的增长率

平均数

现期平均

  • 题型识别：问题时间与材料一致 + 平均（均/每/单位）
  • 计算公式：$ 平均数 = 总数/个数 = A/B $
  • 计算形式：$ 后/前 $
  • 速算技巧：
    • 截位直除
    • 削峰填谷（计算平均数）
      • 定基准，算差距
      • 汇总除以个数，再加上基准

基期平均

  • 题型识别：问题时间在材料之前 + 平均数问法（均/每/单位）
  • 计算公式：$ 基期平均 = A/B *(1+b)/(1+a) $
  • 速算技巧
    • 截位直除
    • 与现期比较

现期平均与基期平均的比较与计算

  • 题型识别：题干中涉及两个时间 + 平均数问法
  • 公式：$ 现期平均-基期平均=A/B-A/B(1+b)/(1+a)=A/B(a-b)/(1+a) $
  • 升降判断
    • a > b，平均数上升
    • a < b，平均数下降
    • a = b，平均数不变
    • a：分子的增长率，b：分母的增长率

平均数的增长率

  • 题型识别：平均数 + 增长 + %
    • 例子：2017年小明家总收入A，同比增长率a；总人口B，同比增长率b，求17年小明家人均收入同比增长了百分之多少？
  • 公式：$ r=(a-b)/(1+b) $（a是分子的增长率，b是分母的增长率）
  • 做题逻辑
    • 确定分子、分母（谁 / 谁）
    • 代入公式：$ r=(a-b)/(1+b) $

倍数

  • 题型识别：时间 + 倍数
    • 例：2017年小明收入10个亿，马云收入2个亿，小明收入是马云的多少倍
  • 现期倍数：$ A/B $
    • 例：2017年小明收入A个亿，同比增长率为a，马云收入B个亿，同比增长率为b，2016年小明收入是马云的多少倍？
  • 基期倍数：$ A/B * (1+b)/(1+a) $
  • 速算技巧：
    • 截位直除
    • 先计算$ A/B $，再看$ (1+b)/(1+a) $与 1 的关系
  • 辨析：
    • A是B的几倍：$ A/B $
    • A比B增长（多）几倍：$ (A-B)/B=A/B-1 $
    • A超过B的n倍：$A > B * n$

特殊增长率

间隔增长率（倍数）

  • 题型识别：中间隔一年，求增长率

    • 例：某企业2015年主营业务收入比2014年的增长率为$ r_1 $，2016年比2015年的增长率为$ r_2 $。

      求：该企业2016年主营业务收入比2014年的增长率为多少？

  • 公式：$ r = r_1 + r_2 + r_1 * r_2 $

  • 解题技巧

    • $ r_1、r_2 $绝对值均小于 10% 时，$ r_1 * r_2 $ 可忽略
    • 结合选型或百化分
    • 若问题是间隔倍数
      • $ 间隔倍数 = r + 1 $

间隔基期量

  • 题型识别：间隔一年，求基期量
  • 公式：$ 间隔基期量 = 现期量/(1+间隔增长率) $
  • 做题步骤
    • 先求间隔增长率 $ r = r_1 + r_2 + r_1 * r_2 $
    • 计算：$ 现期量/(1+间隔r) $

年均增长率

  • 题型识别：年均增长最快、年均增速排序
  • 公式
    • $ (1+r)^n=现期量/基期量 $（n 为现期和基期的年份差）
  • 技巧：
    • 比较：n 相同，直接比较$ 现期/基期 $
    • 计算：放弃

混合增长率

  • 题型识别：部分与总体之间的增长率关系
    • 房产、地产、房地产
    • 出口、进口、进出口
    • 城镇、农村、全国
    • 上半年、下半年、全年
    • 硕士、博士、研究生
  • 判断口诀
    • 混合后居中（大于小的、小于大的）
    • 偏向基期较大的（基期指的基期量，做题中一般用现期量近似代替）

其他

直接找数

  • 注意范围等表述陷阱

简单加减

  • 选项与材料精度相同：尾数法
  • 选项与材料精度不同：估算

排序问题

  • 时间、单位、主题、顺序

综合分析

  • 做题顺序：先CD，再AB，要灵活，复杂的，放最后
  • 陷阱类型：
    • 时间陷阱
      • 时间段是否和材料一致
    • 主题陷阱
      • 主体是否和材料一致
    • 单位陷阱
      • 简单计算时，需要谨慎
    • 概念陷阱
      • 近、将近 => 小于但很接近
      • 超过 => 大于
      • 约 => 可左可右
      • 持续增长、逐年增长 => 每一年都要比上一年增长
      • 增长趋势 => 一般看首尾
      • 说比重，必须要有明显的整体与部分的关系，不能说男生占女生的比重
      • 不足一年 => 当年新增加的
    • 最后注意
      • 选对还是选错，读题时先标出来

常见词语解析

  • 增长率与倍数
    • 增长率：比基数多出的比率
      • A是B的多少倍
      • 倍数：$A/B$
    • 倍数：两数直接的比值
      • A比B多 / 增长多少（%）
      • $ 增长率 = (A-B)/B = A/B - 1 $
    • 倍数 = 增长率 + 1
  • 成数与翻番
    • 成数：几成相当于十分之几
    • 翻番：翻一番为原来的两倍；翻两番为原来的四倍；依此类推，翻n番为原来的$ 2^n $倍
  • 增幅 = 增长幅度 = 增长率 = 增速
  • 降幅：必须下降，r < 0
