    • 法律

      宪法

      • 国徽：中间是五星照耀下的天安门，周围时谷穗和齿轮
      • 中华人民共和国的一切权力属于人民（不是公民）
        • 公民：指具有某一国籍，并根据该国法律享有权利和承担义务的人
        • 人民：是政治概念
      • 自由权是法律权利的核心，是其他权利要素存在的基础

      选举法

      • 不设区的市、市辖区、县、自治县、乡、民族乡、镇的人民代表大会的代表，由选民直接选举

      行政法

      • 行政诉讼法
        • 第三条第三款：被诉行政机关负责人应当出庭应诉。不能出庭的，应当委托行政机关相应的工作人员出庭
        • 第二十六条第二款：经复议的案件、复议机关决定维持原行政行为的，复议机关和原行政机关为共同被告；复议机关改变原行政行为的，复议机关为被告
        • 第二十六条第五款：行政机关委托的组织所作的行政行为，委托的行政机关为被告
      • 撤销行政许可的情形：
        • 行政机关工作人员滥用职权、玩忽职守作出准予行政许可决定的
        • 超越法定职权作出准予行政许可决定的
        • 违法法定程序作出准予行政许可决定额
        • 对不具备申请资格或者不符合法定条件的申请人准予行政许可的
        • 依法可以撤销行政许可的其他条件
## 民法

+ 民事主体的**财产不足以支付的**，优先用于承担**民事责任**
+ **自然人**的行为能力和权利能力**可能分离**，而**法人**的行为能力和权利能力**同时存在**
  + 自然人：公民一出生就享有民事权利，但民事行为能力并不是一出生就具有的。自然人的行为能力分为三种情况：完全行为能力、限制行为能力、无行为能力
  + 法人：法人的民事权利能力和民事行为能力，从法人**成立**时产生，到法人**终止**时消灭

### 民事诉讼法

+ 第九条：人民法院审理民事案件，应当根据**自愿**和**合法**的原则进行**调解**；调解不成的，应当及时判决

### 专利法

+ 第四十二条：发明专利权期限：20年；实用新型专利权和外观设计专利权期限：10年。均自**申请日起**计算

### 经济法

+ 《消费者权益保护法》
  + 第二十二条：经营者提供商品或者服务，**应当**向消费者提供**发票等购买凭证或者服务票据**，消费者索要时经营者**必须**出具。
  + 第二十五条第三款：退款相关，消费者退还**商品应当完好**，经营者应当自**收到商品**之日起**七日内**退还其支付的价款。退还的**运费**由**消费者承担**，二者另有约定按照其约定执行。
  + 第四十三条：消费者在**展销会**等场合消费，若其合法权益收到损害的，可以向销售者或者服务者要求赔偿。展销会等场合**期满后**，消费者也可以向展销会的**举办者**要求赔偿，举办者之后可以向消费者追偿。
  + 第五十五条第一款：经营者有**欺诈**行为的，应当赔偿消费者购买商品或者服务金额的**三倍**，若赔偿金额不到**五百元**的，为**五百元**。

### 合伙企业法

+ **普通合伙企业**由**普通合伙人**组成，合伙人对合伙企业债务承担**无限连带**责任
+ **退伙人**对于基于其退伙前的原因发生的合伙企业债务，承担**无限连带**责任

## 劳动法

+ 劳动合同无效或者部分无效：
  + 以**欺诈**、**胁迫**的手段后者**乘人之危**，是对方在**违背真正意思**的情况下订立或者变更劳动合同的
  + 用人单位免除自己的**法定责任**、**排除劳动者权利**的
  + **违反法律、行政法规**强制性规定的

## 刑法

+ 刑事诉讼法
  + 应当通知法律**援助**机构指派律师为其提供辩护：
    + 盲、聋、哑人，尚未完全丧失辨认或者控制自己能力的精神病人，没有委托人的
  + 被告人**经济困难**或者其他原因没有辩护人的，**本人及其近亲属**可以向法律援助机构提出申请

## 仲裁法

+ 下列纷争**不能**仲裁
  + 婚姻、收养、监护、抚养、继承纠纷
  + 依法**应当由行政机关处理**的行政争议
+ 当事人达成**仲裁协议**，一方向人民法院**起诉**的，人民法院**不予受理**，但仲裁协议**无效除外**
+ 仲裁实施**一裁终局**的制度

# 经济

## 马克思主义政治经济学

+ 资本分类：根据资本在**剩余价值**生产过程中的作用不同，资本可以区分为**可变资本**和**不变资本**。根据**生产资本**的价值周转方式不同，资本可以区分为**固定资本**和**流动资本**
  + 可变资本：指用来购买**劳动力**的那部分资本。它在剩余价值生产中不仅能够创造出劳动力价值，而且能够创造出剩余价值，是一个**可变量**，所以称为可变资本
  + 不变资本：指用来购买**生产资料**的资本
  + 固定资本：指以厂房建筑、机器设备等**劳动资料形式**存在的生产资本
  + 流动资本：指生资本中用于购买原料、燃料、辅助材料等**劳动对象**和用于购买**劳动力**的部分

## 社会主义市场经济体制

+ 区域经济中，**主导产业**和**支柱产业**的作用是不一致的。
  + 主导产业：能够带动一系列**相关产业**发展的产业或者产业集体。
  + 支柱产业：占**国家（地区）**的经济总量的较大比重、对支撑整个**国民经济**既有决定性意义的产业或者产业集体
+ 分配：
  + 初次分配：国民收入首先在**物质生产领域**进行分配
    + 最低工资标准
  + 再分配：国家各级政府以社会管理者的身份主要通过**税收**和**财政支出**的形式参与国民收入分配的过程
    + 最低生活保障
  + 社会保障：
    + 社会保险：**基本**纲领，即五险
    + 社会救助：**最低**纲领，即救灾扶贫
    + 社会福利：**最高**纲领，即福利制度
    + 社会优抚：**特殊**纲领，即军烈属
    + 社会互助：**补充**纲领，即慈善和公益事业
+ 现代企业制度定义为以**市场经济**为基础，以**企业法人制度**为**主体**，以**公司制**为**核心**，以**产权清晰**、**权责明确**、**政企分开**、**管理科学**为**条件的新型企业制度

## 宏观经济

+ 失业：指在一定年龄范围内，**有工作能力**，**愿意工作**并**积极寻找工作**而未能按当时通行的**实际工资水品**找到工作的人

# 管理

## 管理学

### 相关概念

+ 管理地**基本职能**包括：计划、组织、领导和控制

+ 在各种公共资源中，最活跃和最宝贵的财富是**人力资源**
+ 目标管理：以目标为导向，以**人为中心**、以**成果为标准**，促使组织和个人取得最佳业绩的现代管理方法，分为**计划阶段**、**执行阶段**、**检查阶段**、**检讨和改进阶段**。

### 各种概念

+ 效应：
  + 晕轮效应 / 光环效应：对一个人某一特征形成**印象**后，倾向通过此印象**推论**该人**其他方面**的特征
  + 皮格马利翁效应 / 期待效应：基于某种场景的自觉而形成**期望或者预言**，会让该**场景变得适应**这一期望或者预言
  + 首因效应 / 首次效应 / 优先效应 / 第一印象效应：第一印象对今后交往关系的影响，俗称**“先入为主”**
  + 刻板效应 / 刻板印象：对一**群体**形成固定的看法或评价，对该群体中的**某一人**也给予该固定的看法或评价
  + 霍桑效应：人们在意识到自己正在**被关注**或者**观察**的时候，会**刻意改变**一些行为或者是言语表达
  + 马太效应：指**强者愈强**、**弱者愈弱**的现象
  + 刺轮效应：是指人的消费习惯形成后有不可逆性，即**易于向上调整**，**而难于向下调整**
  + 羊群效应：描述经济个体的**从众跟风**心理
+ 理论：
  + 双因素理论：管理学中的**激励机制**原理，双因素即**保健因素**和**激励因素**
    + 保健因素：使职工感到**不满**的，都是属于**工作环境**或**工作关系**方面的
      + 工资
    + 激励因素：使职工感到**满意**的，都是属于**工作本身**或**工作内容**方面的
      + 奖金

## 事业单位相关制度

### 制度相关

+ 根据事业单位经费来源不用，事业单位分为：
  + **全额拨款**事业单位（全供事业单位）：也就是全额预算管理地事业单位，是其所需地事业经费全部由国家预算拨款地一种管理形式
  + **差额拨款**事业单位：按差额比例，财政承担部分，由财政列于预算；单位承担部分，由单位在税前列支，如**医院**等
  + **自收自支**事业单位：作为事业单位地一种主要形式，由于不需要地方财政直接拨款，因而一些地方往往放松对它地管理

### 招聘相关

+ 事业单位需要招聘**外国国籍人员**的，须报**省级**以上政府人士行政部门核准，并按照国家有关规定进行招聘

### 管理相关

+ 机关事业单位在编人员应改变或解除其与单位财政供养关系和人事关系的条件有：
  + 退休
  + 退职
  + 调出
  + 辞去公职
  + 被辞退或解聘
  + 死亡（含死亡宣告）
  + 被开除（含依法被判处刑罚）
  + 法律规定的相关情形
+ 事业单位考核结果：平时考核、年度考核、聘期考核
  + **年度**考核：优秀、合格、基本合格、不合格
  + **聘期**考核：合格、不合格

## 行政管理

+ **行政效率**原则的要求包括：
  + 行政组织精简
  + 行政程序规则化、模式多样化。严格遵循行政**程序**和**时限**，符合行政**规则化**的要求
  + 行政决策的**成本-效益分析**
+ 现在公共行政的**本质内涵**是**以人为本**，而以人为本表现在以下四个方面：
  + 公平、公正--现代公共行政的**价值取向**、**首要目标**
  + 公共服务和公众的**满意度**--现代公共行政的**终极目标**
  + 公共利益--现代公共行政的**诉求对象**
  + 民主、透明--现代公共行政的**基本行为方式**

# 公文

## 基本知识

+ 党政机关公文种类：决议、决定、命令（令）、公报、公告、通告、意见、通知、通报、报告、请示、批复、议案、涵、纪要
+ 公文版记部分：抄送机关、印发机关、印发日期、分数等要素，置于公文最后一页
+ 闭幕词：总结性、评估性、号召性
+ 联合行文：**同级**机关、部门或单位可以联合发文的形式
+ 公文被**撤销**的，视为**自始无效**；被**废止**的，视为**自废止之日起无效**
+ 公文文稿签发前，应当由发文机关**办公厅（厅）**进行审核
+ 公文**成文日期**：
  + 署会议**通过**日期
  + 署机关**负责人签发**的日期
  + 联合发文署**最后签发机关负责人签发**的日期

## 公文格式

+ 公文版头**份号**：
  + 只有**涉密**公文应当标注份号
  + 两篇**不同**的公文份号可能相同
  + 份号应当**顶格编排**，在版心**左上角第一行**
  + 份号一般使用**6位3号**阿拉伯数字，需编**虚位**，如“000001”
+ 主送机关
  + 一般放在**主体**部分
  + 主送机关**过多**，导致首页显示不了正文时，应当将主送机关名称移至**版记**。**抄送机关**都是放在**版记**的
+ 发文机关标志
  + 由发文机关**全称**或者**规范化简称**加’‘文件’‘二字组成
  + 也可以只使用发文机关**全程**或者**规范化简称**
+ 附件说明：
  + 如果多个附件，使用**阿拉伯数字**标注附件顺序号（如：1. xxxxx）
  + 只有**一个**附件时**不需要**使用阿拉伯数字**标注**顺序号
+ 印发机关在**版记**，附注在公文的**主体**

### 古代著作

+ 《资治通鉴》：由**北宋司马光**编著的一部**编年体**史书（以时间顺序），记录**春秋战国**到**宋朝**建立之前，总共**1362**年的历史发展轨迹。
+ 《史记》：由**西汉**史学家**司马迁**撰写的**纪传体**史书，是中国历史上第一部**纪传体通史**（以人物为主），记载了上至上古传说中的**皇帝时代**，下至**汉武帝太初四年**间共3000多年的历史。
  + 史学**双壁**：《史记》与《资治通鉴》
+ 《国语》：是**春秋时期左丘明**所撰的一部**国别体**著作（以国家为单位）。起自**西周穆王**，讫于**战国初年的鲁悼公**，分载周、鲁、齐、晋、郑、楚、吴、越等国约427年的史实。在春秋战国之际由**晋国**的史官编纂成书。是我国**第一部国别体史记**
  + 陈寿的《三国志》也采用了**国别体**体裁

### 宗教

+ 佛教：
  + 起源于古印度，相传为公元前6世纪-前5世纪古印度的**释迦牟尼**所创
  + **藏传佛教**：缘自公元前7世纪，俗称**喇嘛教**
  + **白马寺**：是佛教传入中国后兴建的**第一座官办寺院**，有中国佛教的”**祖庭**“和”**释源**“之称

## 地理

+ 我国的地理分割线
  + **人口密度**分割线：**黑河-云南腾冲**
  + **季风区非季风区**分割线：**大兴安岭、阴山、贺兰山、巴颜喀拉山、冈底斯山**。该线西北为非季风区，东南为季风区。
  + 阶梯分割线：
    + 一、二级阶梯分割线：**昆仑山、阿尔金山、祁连山、横断山脉的东端**
    + 二、三级阶梯分割线：**大兴安岭、太行山、巫山、雪峰山**
  + **秦岭——淮河**一线是一条综合地理界线我国重要的地理分界线，包括**南北方分界线**，**800mm等降水量**，**湿润区与半湿润区界限**，**旱作水田分界线**，**亚热带与暖温带分界线**，**1月平均气温0°C等温线等**。
+ 盆地
  + 柴达木盆地：**矿产**资源丰富，被称为”**聚宝盆**“，主要土类为**盐化荒漠土**和**石膏荒漠土**
  + 四川盆地：地表岩石主要为**紫红色砂岩**和**页岩**，这两种岩石极易风华发育成紫色土，被誉为”**紫色盆地**“
+ 太阳活动 
  + 太阳黑子：
    + 在太阳的**光球层**上发生的一种太阳活动
    + 太阳活动**强弱的主要标志**
  + 太阳耀斑：
    + 发正在太阳**色球层**的爆发现象，也叫**色球爆发**。
    + 太阳活动**最激烈**的显示
  + 太阳风：从**太阳上层大气**射出的**超音速等离子体带电粒子流**
