言语理解与表达

Posted on 2020-02-03 | In 公考 |
  |   WordCount: 1.7k字   |   min2read: 5分钟

言语还是要看自己的语感，还是需要多练题。🤔

中心理解

转折关系

  • 转折之后是重点

  • 典型标志词：虽然…但是、尽管…可是、不过、然而、却、其实/实际上/实际上

  • 非典型性标志词：殊不知、截然不同、截然相反、全新的研究、一种误解、相对而言

  • ps:
    • 错误选项特征：
      • 转折前的内容
      • 例子、原因、背景…
      • 无中生有
      • 绝对表达
      • 与文意相悖
    • 略读句子特征：例子、原因、背景…

主题词

  • 特征：
    • 文段围绕其展开
    • 一般高频出现
    • 多为名词
    • 前有引入或后有解释说明
  • 理论要点：正确选项中需包含文段主题词
  • 干扰选项：范围扩大/缩小/偷换

因果关系

  • 结论是重点

  • 典型格式：因为…所以；由于…因此…

  • 因果标志词：
    • 所以、因此、因而、故而、于是、可见、看来（为什么、为何）
    • 导致、致使、使得、造成
  • 文段特征
    • “因此”句在结尾：多为文段中心句
    • “因此”句在开头/中间：
      • 后为进一步解释说明：“因此”句为中心句
      • 后有其他关联关系：需结合多种关联词共同分析

必要条件关系

  • “只有…才”中间是重点
  • 典型格式：只有…才…
  • 对策标志词：
    • 应该、应当、必须、需要、亟须、亟待 + 做法
    • 通过/采取…手段/途径/措施/方式/方法/渠道，才能…
    • 呼吁、倡导、提倡、提醒、建议 + 做法
    • 前提、基础、保障、关键
  • 文段特征：
    • 对策在结尾：提出问题 + 分析问题 + 解决问题
    • 对策在中间：提出问题 + 解决问题 + 解释说明（意义效果）
    • 对策在开头：对策 + 正方面论证/原因论证
  • 反面论证：
    • 典型格式：如果/倘若/一旦… + 不好的结果
    • 具体应用：把前面的做法反过来，即为正确答案
    • 常见错误选项特征：假设变成现实的表述
  • 提示：
    • 文段只出现提出问题 + 分析问题，解决问题可能出现在选项中作为正确答案
    • 对策不万能，一定要注意

程度词

  • 标志词：更、尤其、正是、特别是、真正、根本、最（核心、突出）等
  • 要点：程度词所在的语句通常为重点

并列关系

  • 概括完整全面
  • 文段特征：
    • 包含并列关联词及表点，如此外、另外、统计、及时，“；”
    • 句式相同或相近
    • 无明显其他关联词语
  • 选项特征：
    • 两方面情况：和、及、与、同
    • 更多情况：许多、一些、不同、各种、一系列
  • 提示：
    • 分局表达意思相同：提取共性
    • 表述片面的选项需排除

行文脉络

  • 要点：把握中心句及分述句的特点
  • 中心句特征：观点
    • 对策
    • 结论
    • 评价
  • 分述句特征：
    • 举例子：“比如”、“例如”、“…就是例证”等
    • 调差报告、数据资料等
    • 正反论证
    • 原因解释
    • 并列分述
  • 提示：选项陷阱总结
    • 非重点
    • 无中生有

细节判断题

  • 提问方式：
    • 以下对文段理解正确/不正确的是
    • 符合/不符合这段话意思的是
    • 从文段中可以得知/推出的是
  • 错误选项类型
    • 无中生有
    • 偷换概念
      • 替换
      • 混搭
    • 偷换时态
      • 将来时（将要、立刻、趋势、以后）
      • 完成时（已、已经、了、完成）
      • 进行时（正在、在…中、着）
  • 细节主旨化：优选契合主旨的选项

语句表达

语句排序

  • 解题要点：从选项入手
    • 确定首句
    • 确定捆绑集团、确定顺序、确定尾句
  • 解题方法
    • 确定首句：
      • 首句特征：
        • 下定义：…就是/是指
        • 提出观点：有人说、人们普遍认为、…认为
        • 背景引入：随着、近年来、在…大背景下/环境下
      • 非首句特征：
        • 指代词：这、那、他、其、该、此
        • 关联词后半部分：但、却、而且
    • 确定捆绑集团：
      • 指代词：这、那、他、其、该、此。联系指代词对象，找出上文。
      • 关联词：
        • 配套出现确定捆绑
        • 单个出现分析句意
    • 确认顺序
      • 时间顺序
      • 逻辑顺序：
        • 观点 + 解释说明
        • A 和 B
    • 确认尾句：
      • 因此、所以、看来、这…引导的总结性尾句
      • 对策

语句填空题

  • 解题要点：注意填空位置
  • 横线在结尾：
    • 总结前文
    • 提出对策
  • 横线在开头：概括文段的中心内容
  • 横线在中间：
    • 注意与上下文的联系
    • 把握好主题词，保证与文段话题一致。

接语选择题

  • 提问方式：作者接下来最可能讲述的是？
  • 理论要点：重点关注文段最后一句话
  • 干扰选项：文段中已经论述过的内容
  • PS：理解要紧紧依托文段，不能无中生有

逻辑填空

  • 词的解析：
    • 词义侧重：
      • 第一种：拆字组词、用不一样的字单独组词
      • 第二种：整词搭配、用整个词进行组词搭配
    • 固定搭配：
      • 搭配一个词：瞻前顾后找准搭配对象
      • 搭配多个词：
        • 横线所填词语搭配由“和、及、与”引导的并列结构
        • 需与并列结构搭配恰当
    • 程度轻重：
      • 区分词的程度轻重：
        • 程度轻：如大相径庭：相差很大、相差很多
        • 程度重：如截然不同：完全相反、完全不同
      • 要点：所填词的程度与文段意思的轻重保持一致
    • 感情色彩：
      • 区分词的感情色彩：
        • 褒义：成果
        • 中性：结果
        • 贬义：后果
      • 要点：所填词的感情色彩要与文段的感情色彩保持一致
  • 文段逻辑
    • 转折：
      • 标志词：但是、可是、然而、却、其实、实际上等
      • 要点：前后词义相反
    • 递进：
      • 标志词：而且、更、甚至、乃至、遑论等
      • 要点：语义程度前轻后重
  • 语境分析
    • 关联关系
      • 转折：
        • 标志词：但是、然而、却等
        • 理论要点：前后语义相反
      • 递进：
        • 标志词：而且、甚至、更等
        • 理论要点：语义程度前轻后重
      • 并列
        • 标志词
          • 同义并列：顿号（、）、逗号（，）
          • 反义并列：不是…而是…、是…不是…、相反、反之等
        • 理论要点：同意并列连接前后语义相近；反义并列连接前后语义相反
        • 提示：相同语句也可以表示并列
    • 对应关系
      • 解释类对应
        • 题干特点：分句，____，分句
        • 标志词：即、就是、可以说、无异于、比如、例如等
        • 标点：冒号（：）、破折号（——）
      • 重点词句对应：
        • 重点词：指代词、主题词、形象表达
        • 重点句：完整语句
      • 提示：找准对应关系，要基于对整个文段意思的理解与把握
