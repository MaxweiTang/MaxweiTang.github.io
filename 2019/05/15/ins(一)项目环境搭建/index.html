前言

之前对于知识的学习感觉太偏重于理论了，前段时间做了个类似（抄袭）instagram 的图片分享网站来对之前学习的知识进行巩固，同时想在这个小项目中加强自己对业务的理解。那么这个小项目前端是采用 Vue 框架，后台呢是基于 Node ，数据库采用的是 NoSQL 数据库 – MongoDB。这个系列（坑）呢主要是想记录下我在这个项目中遇到的各种问题以及解决方法，同时希望能让我养成总结的好习惯，加深记忆和理解。

二、MongoDB

MongoDB 是一个基于分布式文件存储的数据库。由 C++ 语言编写。旨在为 WEB 应用提供可扩展的高性能数据存储解决方案。MongoDB 是一个介于关系数据库和非关系数据库之间的产品，是非关系数据库当中功能最丰富，最像关系数据库的。

同样关于 MongoDB 的安装教程网页也有很多详细的教程，这里也就直接说下大概的流程吧。

1、下载安装

下载直接在 MongoDB 官网下载即可 https://www.mongodb.com/download-center#community

安装根据其引导安装即可。

当在命令行中输入 mongo -version 能够显示时及安装成功

2、启动 MongoDB

在命令行中输入 mongod

之后在开启一个新的命令窗口 输入 mongo ，就能打开MongoDB 的 shell 窗口

在 shell 中输入show dbs 就能显示出当前已存在的数据库

3、MongoDB 可视化工具

当然 MongoDB 也有可视化工具，我使用的是 Robo 3T

官网地址：https://robomongo.org/

进入官网后 直接下载 Robo 3T 即可

这里我下载的是绿色版，解压后直接使用，看个人喜好啦。

下载完成后直接连接使用即可
